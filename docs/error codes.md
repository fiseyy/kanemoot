## Система логирования

Разряд              | Что обозначает                      
------------------- | ------------------------------------
`AA` (старший байт) | Категория                           
`BB`                | Код ошибки в категории              
`CC`                | Модуль / подсистема                 
`DD` (младший байт) | Доп. информация / instance ID / флаг

## Расшифровка разрядов:

### `AA` — Категория
- 01 - UI  
- 02 - Account  
- 03 - System (файлы, хранилища, логгер)  
- 04 - Network  

### `BB` — Код ошибки в категории

#### UI
- 01 - Ошибка с QML Engine
- 02 - Ошибка компонента QML
- 03 - Не удалось создать QML-страницу
- 04 - Недопустимый объект страницы или окна
- 05 - Не удалось создать объект темы
- 06 - Объект root не инициализирован
- 07 - QML не загружен
- 08 - Не удалось загрузить QML
- 09 - Не удалось вызвать сигнал QML
- 0A - Объект, необходимый для работы UI, не инициализирован


#### Account:
- 01 - Неверный пароль  
- 02 - Пользователь заблокирован  
- 03 - Аккаунт не найден 
- 04 - Имя пользователя занято
- 05 - Почта занята

#### System
- 01 - Ошибка инициализации незащищённого хранилища
- 02 - Ошибка инициализации защищённого хранилища
- 03 - Не удалось открыть файл лога для записи
- 04 - Не удалось переименовать файл

#### Network
- 01 - Нет связи с сервером
- 02 - Неверный формат ответа сервера
- 03 - Подключение уже выполняется
- 04 - Сервер временно недоступен
- 05 - Ошибка шифрования
- 06 - Неизвестный сервис



### `CC` — Модуль
- 01 - AppController
- 02 - ChatPage
- 03 - LoginPage
- 04 - RegistrationPage
- 05 - ApiEndpoints
- 06 - App
- 07 - ErrorHandler
- 08 - SecureStorage
- 09 - SettingsManager
- 0A - SystemFatalError
- 0B - AuthManager
- 0C - WebSocketClient
- 0D - Logging


### `DD` — Доп. информация / instance ID / флаг (опционально)
- 00 - стандартный случай  
- 01 - особый кейс или расширенная информация
